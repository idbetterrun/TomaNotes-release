## 🌟 Introduction / 项目简介

**TomaNotes** is a next-generation desktop note-taking application designed for users who prioritize privacy and professional writing experiences. Built with Electron and React, it bridges the gap between powerful rich-text editing and robust security.

**TomaNotes** 是一款专为重视隐私和专业写作体验的用户打造的下一代桌面笔记应用。基于 Electron 和 React 构建，它完美融合了强大的富文本编辑功能与坚固的安全性。

---

## ✨ Key Features / 核心特性

### 🔒 Security First / 安全至上
* **Global PIN (6-digit):** Protect your entire application with encrypted master password.
    * **全局 6 位 PIN 码：** 使用加密的全局主密码保护整个应用。
* **Biometric Auth:** Native macOS Touch ID integration for instant access.
    * **生物识别：** 集成 macOS 原生 Touch ID，实现秒速验证。
* **Note-level Encryption:** Dual-layer security for sensitive information.
    * **单篇笔记加密：** 为敏感信息提供双重安全保障。

### ✍️ Professional Editor / 专业编辑器
* **Hybrid Engine:** Seamlessly switch between **Rich Text (Tiptap)** and **Markdown (CodeMirror)**.
    * **双引擎：** 在 **富文本 (Tiptap)** 与 **Markdown (CodeMirror)** 之间无缝切换。
* **Native Experience:** Fully integrated macOS-style context menus.
    * **原生体验：** 深度集成 macOS 风格的全功能右键菜单。
* **Focus Mode:** Collapsible sidebar for an immersive writing experience.
    * **专注模式：** 可折叠侧边栏，提供沉浸式写作环境。

### 🇨🇳 Localization / 本土化适配
* **Chinese Typography:** Support for traditional Chinese font sizes (e.g., "号字制").
    * **中文排版：** 深度适配大中华区熟悉的“号字制”字号调整。
* **Full i18n:** Bilingual support for Simplified and Traditional Chinese.
    * **全界面汉化：** 完善的简繁中文支持。

## 🚀 Tech Stack / 技术栈

* **Framework:** [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
* **Runtime:** [Electron](https://www.electronjs.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Editor:** [Tiptap](https://tiptap.dev/) & [CodeMirror 6](https://codemirror.net/)
* **Animation:** [Framer Motion](https://www.framer.com/motion/)
